⚙️ System methodology (brief)

Our platform integrates four specialised agents orchestrated by a debate‑driven coordinator:

  • 🔎 Query Agent – Decomposes user intent into 4 stance dimensions (official, support, oppose, neutral) + credibility scoring + near‑duplicate filtering.
  • 🎥 Multimodal Agent – Built on LangGraph; downloads short‑form video, extracts keyframes (OpenCV), generates captions via VLM, and fuses text+visual sentiment. ⚠️ Image recognition module under active refinement.
  • 🗣️ Agent Coordinator – Monitor + Moderator + Reader: implements iterative debate rounds to cross‑validate findings and mitigate bias.
  • 📊 Report Agent – Template‑based report builder with modular chapters, knowledge graph, and interactive export (PDF/HTML).
📌 Current implementation focus – The Multimodal Agent (LangGraph pipeline) successfully processes video keyframes and performs sentiment classification on captions. The team is debugging the image recognition pipeline to improve accuracy for standalone images and visual sentiment cues.
